    类型一：建筑工签骗局（重点高发）

    以「爱尔兰建筑工高薪」为卖点，吸引有建筑经验的工人。实际普通建筑工人（general labourer）不在爱尔兰紧缺职业清单上，极难合法获得工签。

    • 核心手法：中介声称「建筑工缺人，来了就能上岗」，实际利用信息差将语言签/学生签（Stamp 2A）包装成「建筑工签」
    • Safe Pass 误导：中介宣称帮忙办安全证（Safe Pass，1天课程120欧）就能合法工作——但 Safe Pass 只是入场证，不是工作签证
    • 录视频当龙套：部分中介要求受害者录制虚假「工作视频」，用于欺骗后续国内客户
    • 落地即失业：多篇笔记证实，建筑工人到达爱尔兰后被雇主立即解雇，签证随之失效，进退两难
    • 半死不活的建筑公司：没有工程的建筑公司靠工签业务赚得盆满钵满，受害者为找其他方式留下来不敢发声

    典型案例：「高小歪歪」详细拆解了建筑工签骗局（33赞31评论），明确指出普通工地工人不在紧缺清单上。「爱尔兰白桃汽水PureBai」揭露爱尔兰建筑工被落地解雇。「风自在」直言「建筑工签基本来了就没工作，自己找工，非常不稳定」。

    损失程度：中介费 5-15 万人民币，到达后面临无工可做、签证失效、被遣返的三重困境。

    类型二：肉厂工签骗局

    多篇笔记反复提及，是爱尔兰工签骗局中受害人数最多的类型。

    • 国内黑中介以「爱尔兰正规肉厂工作」「高薪剔骨工」为噱头，通过社交平台大量发布广告
    • 承诺「包工作、包住宿、月薪高」，收取高额中介费（5万+ 人民币）
    • 受害者到达爱尔兰后：工资低于最低标准、住宿条件恶劣、被克扣费用
    • 肉厂雇主与中介分成后，故意找茬、施压，甚至解雇工人

    类型三：关键技能工签（Critical Skills）骗局

    利用爱尔兰「关键技能工签」政策，声称可提供紧缺职业清单上的岗位担保，实际岗位不存在或与宣传不符。部分案例中雇主在工人落地后短期内解雇，导致签证失效。

    类型四：语言签伪装工签骗局

    中介以「语言学校 + 工作」为卖点，实际办理的是语言学习签证（不可合法工作）。部分中介甚至要求受害者录制虚假「工作视频」用于欺骗后续客户。

    类型五：虚假中介/杀猪盘

    国内社交平台上大量 IP 在国内的账号，伪装成「爱尔兰华人」「工签成功者」发布经验帖，私信诱导咨询，收取费用后消失或提供虚假材料。部分从2018年至今持续运营黑产。

    8.11 高危信号识别清单

    1. 中介IP在国内，声称「在爱尔兰」 — 实际不在当地，无法提供真实保障

    8.13 规避骗局实操指南

    第一步：出发前核查（必做）

    1. 查雇主注册信息 — 登录爱尔兰 CRO 官网（cro.ie），输入雇主公司名，确认公司真实注册、正常运营。查不到的 = 不存在。
    2. 核实工签类型 — 访问爱尔兰 DJEI 官网，确认你的岗位是否在紧缺职业清单上。普通建筑工人、普通工厂工人大概率不在清单上。
    3. 要求看正式合同 — 出发前必须签署英文雇佣合同，合同中需明确：薪资、工时、岗位、地点。拒绝签署合同的 = 大概率骗局。
    4. 确认签证类型 — 拿到签证后仔细看签证类型。Stamp 2 / Stamp 2A 是语言学习签，不允许全职工作。
    5. 计算真实成本 — 正规雇主担保工签费用由雇主承担。如果你需要自己掏 5万+，大概率有问题。

    第二步：识别骗局话术（遇到就跑）

    骗局话术真相
    "建筑工缺人，来了就能上岗"普通建筑工不在紧缺清单，极难合法获得工签
    "读3个月语言学校就能转工签"语言签(Stamp 2A)不允许工作，无法直接转工签
    "包工作、包住宿、月薪高"承诺越完美，骗局概率越高
    "名额有限，马上就要涨价了"制造紧迫感 = 典型销售话术
    "帮你办个Safe Pass就能上岗"Safe Pass只是安全培训证，不是工作签证
    "先录个视频，以后用得着"录视频是用来骗下一批受害者的
    "中介费只要5万，比投资移民便宜多了"正规工签不需要工人掏钱

    第三步：到达后自我保护

    1. 保留所有证据：聊天记录截图、转账凭证、签署的合同、对方承诺的录音
    2. 第一时间核实工作条件：到岗后如果发现与合同不符，立即记录并拍照
    3. 不要签署不明文件：尤其是不理解内容的英文文件
    4. 不要交出护照：任何以"帮你办手续"为由收走护照的行为都是违法的

    第四步：被骗后怎么办
